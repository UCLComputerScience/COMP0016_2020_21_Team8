AvaBot User Manual

We believe that the best way for users to learn how to use AvaBot is to actually use it!
Here are some tips:
1. It may take some time for AvaBot to connect to the web chat, it'll send you a msg as soon as it's ready.
2. Choose which feature to use by clicking the buttons provided by AvaBot.
3. It may also take some time for AvaBot to process a document.
4. AvaBot will guide you through everything so just follow.
5. Enjoy!
